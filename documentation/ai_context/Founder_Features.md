# Founder File: Features & Branding

## Brand Identity
- **App Name**: "Yamparala Dev App"
- **App Logo**: Custom brand icon located at `public/images/app-logo.png`.
- **Target User**: Lead Design Engineers and their teams.
- **Admin**: Yamparala Rahul (Lead Design Engineer).

## Core Feature Modules

### 1. Unified Dashboard (Logs Tab)
- **Design Log Tracking**: Capturing design decisions, research, and progress.
- **Visual Evidence**: Support for multiple high-quality image uploads per log.
- **Timeline View**: Chronological view of design evolution.
- **Waste Management**: Soft-delete Trash bin for accidental deletions.

### 2. Knowledge Base (Wiki Tab)
- **Library of Excellence**: Shared documentation and design guides.
- **Media Ready**: Embed YouTube videos directly to demonstrate prototypes or research findings.
- **Collaborative Discussion**: Comments system on every page for team feedback.

### 3. Resource Library (Resources Tab)
- **Toolbox**: Shared links to tools, inspiration, and external assets.
- **Smart Previews**: Automatic screenshot generation for every link shared.
- **Team vs. User Sections**: Distinction between official team resources and individual contributions.

### 4. Direct Support (UPI Integration)
- **Monetization/Support**: Direct UPI payment integration for users to support the developer.
- **QR & Intent**: Support for both mobile app deep-linking and desktop QR code scanning.
